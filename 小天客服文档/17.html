3、启动以及调试

注意代码目录之间的切换 xiaotian 和 xiaotian-socket

1、开启websock模式，xiaotian/config/seller.php

// socket.io 还是 websocket 模式
// 1: socket.io 2: websocket
'model' => 2

2、配置基础信息

打开xiaotian-socket/config.php

'version' => '1.0',

// websocket 对外服务端口
'ws_port' => 2020,

// http api 服务对外端口
'api_port' => 2945,

// gateway worker number
'gateway_worker' => 8,

// business worker number
'business_worker' => 8,

// 是否开启客服只允许单点登录 0 不开启 1 开启,
// 切换需要重启
'single_login' => 1,

// 是否开启 ssl
'is_open_ssl' => false,

'context' => [
    'ssl' => [
        // 请使用绝对路径
        'local_cert'  => '磁盘路径/server.pem', // 也可以是crt文件
        'local_pk'  => '磁盘路径/server.key',
        'verify_peer'   => false
    ]
],

'database' => [
    'host'        => '127.0.0.1',
    'user'        => 'root',
    'password'    => 'root',
    'database'    => 'xiaotian_v2',
    'port'        => '3306',
    'charset'     => 'utf8',
]
参数 说明
ws_port 对外服务的websocket端口（需打开防火墙）
api_port api模式服务端口 （需打开防火墙）
gateway_worker 负责处理用户连接的进程，建议设置成cpu核数 或者 2-4倍
business_worker 负责处理业务的进程数，建议设置成cpu核数
database 数据库连接配置

3、配置连接信息

打开 xiaotian/config/xiaotian.php

// 通信协议
'protocol' => 'ws://',

// socket server
'socket' => '127.0.0.1:2020'

socket 配置对应的服务器的ip (如果您是https，这里的配置参考 如何开启https) 和 你设置的ws的端口

4、启动gatewayworker

进入 xiaotian-socket windows下点击目录下的 start_for_win.bat linux下运行

php start.php start

此是debug模式 其他命令:

php start.php start  // debug执行
php start.php start -d // 守护执行
php start.php restart -d // 重启守护执行
php start.php stop // 停止执行

为了增强系统稳定性和提高系统并发性能，强烈建议您安装 event扩展

yum install libevent -y  
wget https://pecl.php.net/get/event-2.3.0.tgz  
tar -zxvf event-2.3.0.tgz  
cd event-2.3.0  
/www/server/php/70/bin/phpize  
./configure --with-php-config=/www/server/php/70/bin/php-config  
make && make install  
echo "extension=event.so" >> /www/server/php/70/etc/php.ini

这里的 /www/server/php/70/bin/phpize 路径，根据您的实际路径做出修改，比如我的路径是 /www/server/php/72/bin/phpize 那么里面的所有的关于php文件的路径我全部是 /www/server/php/72/***
